Primair font

Als primair font wordt Montserrat gebruikt, een gratis te gebruiken Google Font. Het wordt gebruikt voor alle teksten.

Alle stijlen voor het primaire font zijn geprefixed met @font_primary. Er worden er geen font weights gebruikt. Voor vetgedrukte tekst gebruik je dus font-family: @font_primary_body.

Aa Montserrat Light @font_primary_light
Aa Montserrat Regular @font_primary
Aa Montserrat Semibold @font_primary_semi
Aa Montserrat Bold @font_primary_bold

Secundair font

Als secundair font wordt Leage Gothic gebruikt, een gratis te gebruiken lettertype. Het wordt gebruikt voor koppen en accentteksten.

Voor het secundaire font gebruik je de variabele @font_secondary. Dit lettertype wordt in koppen vaak UPPERCASE gebruikt.

Aa League Gothic @font_secondary
