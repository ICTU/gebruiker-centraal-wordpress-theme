Use cards for content

Standaard card

Default template voor een kaart. Kan gebruikt worden bij een verwijzing naar bijvoorbeeld gewone pagina. Mag over de gehele breedte wanneer hij niet in een grid staat.

Card met afbeelding

Template voor een kaartje met een afbeelding. Deze krijgt modifier class "card--with-image" mee.

Card met metadata

Template voor een kaartje met een afbeelding én metadata. Metadata heeft zijn eigen component - die kan overal worden gebruikt waar verschillende items als metadata worden getoond.
